# Evals

Use evals to improve the skill.

## Test Prompt Types

- Rough outline to deck plan.
- Source-only deck with no hallucination.
- Codebase architecture deck.
- Research-enabled deck with citations.
- Rewrite a deck for concision.
- Repair one visually crowded slide.

## Objective Checks

- Required artifacts exist.
- Research mode is respected.
- Slide sorter has one message per slide.
- Claim ledger contains no unsupported numbers.
- Code slides cite files or snippets.
- Theme selection includes a light option when audience is broad or conservative.
- Repair prompts do not rewrite unrelated slides.

## Qualitative Checks

- Story is clear.
- Visual aids make hard ideas easier.
- Notes sound speakable.
- Design direction fits the audience.
- Output feels specific to the provided material.
