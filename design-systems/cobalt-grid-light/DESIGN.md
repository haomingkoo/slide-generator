# Cobalt Grid Light

Use for technical, hackathon, product strategy and proposal decks that need to feel sharp, modern and visual without becoming dark or corporate.

## Palette

- White or ice-blue graph-paper canvas.
- Electric cobalt as the main accent.
- Black or near-black text for contrast.
- Cyan, mint or cool grey as secondary accents.
- Avoid cream, beige, parchment and bone-paper backgrounds unless the user explicitly asks for them.

## Layout

- Fixed 16:9 stage with a visible grid.
- Large point-first titles, preferably one line when possible.
- Hairline rules and axis labels instead of decorative card stacks.
- Metric blocks, timelines and tables should align to the grid.
- Use one strong visual field per slide: graph plane, stepped diagram, data strip, screenshot panel or comparison table.

## Visuals

- Cobalt graph grid.
- Stair-step markers.
- Pixel or signal ticks used sparingly.
- Thin divider rules.
- Compact evidence tags and source lane.

## Known Failure Modes

- Italic display type can hurt readability if overused.
- Grid decoration can compete with real tables.
- Cobalt becomes harsh if every element is saturated.
- Pixel-glitch accents should not touch source text, footers or presenter controls.
