# Clean Surgical Light

Use for academic, consulting, classroom, executive, and broad-audience decks.

## Palette

- Background: tinted white or cool off-white
- Text: near-black
- Secondary text: neutral gray
- Accent: one restrained color
- Semantic colors: green for good, amber for caution, red for risk, blue for neutral system

## Typography

- Sans-serif.
- Strong title, calm body.
- No tiny labels.
- Avoid loud display type unless the slide is a deliberate section break.

## Layout

- Visible grid.
- Generous whitespace.
- One main visual per slide.
- Thin rules and dividers.
- Cards only when they structure repeated information.

## Motion

- Minimal.
- Use fade, reveal, or small translate.
- No bouncing or decorative motion.
