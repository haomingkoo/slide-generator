# Cool Campaign Light

Use for campaign strategies, go-to-market plans, marketing proposals, launch plans and lightweight strategy decks where the audience needs a crisp plan, not a dense consulting report.

## Palette

- White or very light grey background.
- Lavender accent for emphasis and section tags.
- Cool silver/blue image fields when a visual identity is useful.
- Deep charcoal text for legibility.

## Layout

- Large calm titles, but keep them short enough that proof stays visible.
- Use numbered rows for challenge, risks and next actions.
- Use KPI strips for success metrics.
- Use chart plus allocation ladder for budget and channel planning.
- Use split image/text layouts as rhythm, not as proof.

## Visuals

- Campaign timeline.
- Budget allocation chart.
- Persona card.
- KPI strip.
- Ask and next-step rows.

## Known Failure Modes

- Generic metallic textures can become mood-board filler.
- Lavender can become one-note if every element uses the same hue.
- Timeline cards become crowded when every milestone contains a paragraph.
- Categorical budgets should not be shown as line charts unless there is a real time series.
