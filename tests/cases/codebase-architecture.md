# Eval Case: Codebase Architecture Deck

## Prompt

Review the provided codebase and create a 7-slide technical architecture deck. Include one Mermaid runtime diagram, one architecture map, and two short code snippets that prove the core mechanism. Use source-only mode.

## Expected Output

- Codebase review.
- Architecture map with nodes, edges, and evidence.
- Mermaid diagram from actual code.
- Code snippets with file path and line evidence.
- Audit report that flags any inferred architecture.

## Key Checks

- No invented services.
- Diagram edges cite files or are labeled as inference.
- Snippets are short and relevant.
