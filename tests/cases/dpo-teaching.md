# Eval Case: DPO Teaching Deck

## Prompt

Turn the DPO interactive demo material into a 10-slide teaching deck for classmates who understand basic machine learning but have not studied RLHF. Use source-only mode. Include speaker notes. Use a clean light theme unless the user explicitly asks for dark mode.

## Expected Output

- Intake brief.
- Claim ledger with DPO paper claims separated from analogies.
- Story spine that moves from preference problem to DPO simplification.
- Visual aid plan with response comparison, RLHF vs DPO pipeline, probability shift, and math stepper.
- Slide specs with speaker notes.

## Key Checks

- No unsupported benchmark numbers.
- DPO does not get described as magic.
- Math appears after intuitive explanation.
- Speaker notes sound natural.
