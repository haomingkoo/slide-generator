# Code Snippet Slide

Use a code snippet only when code is evidence.

## Rules

- Show 6 to 12 lines.
- Use actual code from the repo.
- Include file path and line numbers.
- Highlight one mechanism.
- Explain why the snippet matters.

## Avoid

- boilerplate,
- long functions,
- generated code without context,
- snippets that are only decorative.
