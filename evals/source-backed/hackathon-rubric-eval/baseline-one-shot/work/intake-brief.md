# Intake Brief

## Deck Job

- Audience: builders preparing for a hackathon judging session.
- Purpose: teach a short demo pitch structure.
- Format: three-minute live presentation plus Q&A.
- Target: six slides.
- Output: HTML review deck with speaker notes.

## Baseline Method

This is a one-shot-style baseline fixture, not the skill workflow output. It
uses the same source pack and claim ledger as the skill run, but it does not add
the full artifact-first planning loop before slide writing.

## Constraints

- Use source-backed claims only.
- Avoid universal "winning formula" language.
- Keep project-specific metrics, traction, and model claims out of the generic
  teaching deck.
