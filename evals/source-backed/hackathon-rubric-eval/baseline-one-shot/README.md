# One-Shot Baseline Fixture

This project is a frozen one-shot-style baseline for the hackathon rubric eval.

It uses the same brief, source bundle, claim ledger, theme, and quality rubric as
the skill workflow output one directory up. The slide specs are intentionally
more generic so the comparison can test whether the artifact-first workflow adds
measurable structure: rubric routing, demo proof, fallback guidance, audience
questions, and visual purpose.

This is not a universal benchmark. Replace this fixture with a captured one-shot
model output when running a model-specific eval.
