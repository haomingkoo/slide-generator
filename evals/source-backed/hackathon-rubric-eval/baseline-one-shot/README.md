# Structural Proxy Baseline Fixture

This project is a frozen self-authored structural proxy baseline for the hackathon rubric eval.

It uses the same brief, source bundle, claim ledger, theme, and quality rubric as
the skill workflow output one directory up. The slide specs are intentionally
more generic so the comparison can test whether the artifact-first workflow adds
measurable structure: rubric routing, demo proof, fallback guidance, audience
questions, and visual purpose.

Despite the directory name, this is not a captured one-shot model output. Do not
cite it as proof that the skill beats one-shot generation. Use it as a regression
harness for structural discipline, then replace it with captured one-shot model
output when running a model-specific eval.
